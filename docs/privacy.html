CivicPlus Internal Toolkit

Privacy Policy

Privacy Policy

Effective date: July 16, 2026

CivicPlus Internal Toolkit is a Chrome extension for authorized CivicPlus employee use. It provides workflow helpers for CivicPlus CMS/admin/editor pages, Design Center, Live Edit, and user-approved CivicPlus customer vanity domains.

The extension is intended for internal CivicPlus workflows. It does not sell user data, use user data for advertising, or send saved toolkit data to third-party analytics services.

Data the Extension Handles

The extension may access website content on CivicPlus CMS/admin/editor pages to provide its features. This may include:

  • CMS page content and editor-page data
  • Theme and style configuration
  • Widget skin data
  • Snippets, buttons, links, and related CMS configuration
  • Module/page content needed for employee-invoked toolkit workflows

The extension stores toolkit settings, enabled/disabled tools, saved snippets, saved skins/buttons, trusted vanity-domain preferences, and local workflow preferences in Chrome extension storage.

How Data Is Used

Data is used only to provide the extension's employee workflow features, such as applying styles, saving reusable snippets, importing/exporting CMS configuration, running user-invoked CMS helper tools, and maintaining local toolkit preferences.

Data Sharing

The extension does not sell user data. The extension does not use user data for advertising. The extension does not send saved toolkit data to third-party analytics services.

Remote Code

The extension does not fetch or execute remote JavaScript. Extension code is bundled with the extension package.

Data Retention

Data stored by the extension remains in the user's local Chrome extension storage until the user deletes it, clears extension data, or uninstalls the extension.

Contact

For questions about this extension, contact the publisher listed on the Chrome Web Store item or the CivicPlus internal team responsible for the toolkit.
