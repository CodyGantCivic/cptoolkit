Getting Started

The CivicPlus internal toolkit contains several different tools. Information about each one will be in its section below.

Main Menu

The main menu is shown when clicking the icon at the top right corner of the browser. It contains checkboxes to enable or disable any of the tools in the toolkit.

Settings

You can open the settings by right-clicking the Main Menu () icon and selecting Options. The settings page will allow you to enable or disable any of the tools in the toolkit, and will also allow you to pick additional options for each tool.

On-Load Tools

These tools run automatically when you visit CivicPlus sites.

On-Demand Tools

These tools can be triggered from the context menu (right-click) on compatible pages.
