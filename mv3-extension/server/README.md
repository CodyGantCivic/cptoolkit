# CP Toolkit MCP Server

Local server for capture files generated by the CivicPlus Toolkit extension.

## What it provides

- HTTP collector endpoint for the extension:
  - `POST /collect`
  - `GET /captures`
  - `GET /captures/:id`
  - `GET /health`
- MCP tools over STDIO:
  - `list_captures`
  - `get_capture`
  - `find_requests`
  - `trace_request`

## Install and run

```bash
cd server
npm install
npm start
```

Default HTTP URL: `http://localhost:9001`

Set a different port:

```bash
CP_MCP_HTTP_PORT=9010 npm start
```

Disable HTTP collector and run MCP-only STDIO:

```bash
CP_MCP_DISABLE_HTTP=1 npm start
```

## Capture storage

Captures are stored as JSON files in:

`server/captures/`

The server applies a basic secret redaction pass before writing files.

## Extension flow

1. In extension popup: click `Start Capture`.
2. Reproduce the CMS issue.
3. Click `Export JSON` (local file) or `Send` (uploads to `/collect`).
4. Use MCP tools against stored captures.

## Codex MCP wiring

Use this process command in your MCP client config:

```json
{
  "command": "node",
  "args": ["C:/path/to/mv3-extension/server/index.js"]
}
```

Adjust the path to your local workspace.
